# AntSport Shopify Theme

Tema Shopify completo con homepage popolata e fallback Liquid.

## Importante
Questa versione include sia:

- `templates/index.json`
- `templates/index.liquid`

Così la homepage viene caricata correttamente anche se Shopify non aggancia il template JSON.

## Upload
Vai in Shopify:

Online Store → Themes → Add theme → Upload zip file

Carica `antsport-shopify-theme-fixed-home.zip`.

## File principali
- `layout/theme.liquid`
- `templates/index.liquid`
- `templates/index.json`
- `sections/main-home.liquid`
- `assets/antsport.css`
- `assets/antsport.js`
